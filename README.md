# Праздничный вечер — сайт-приглашение

Электронное приглашение на праздничный вечер в честь бракосочетания Ярослава и Анастасии. Одностраничный сайт в винтажном летнем стиле: тёплая палитра, ботанические иллюстрации, минималистичная типографика.

## Стек

- Vite + React + TypeScript
- Tailwind CSS v4

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Собранные файлы появятся в `dist/` и готовы для размещения на любом статическом хостинге (Netlify, Vercel, GitHub Pages).

## Структура

- `src/data/content.ts` — весь текст приглашения, а также ссылка на карту (`mapUrl`) и ссылка для подтверждения присутствия (`rsvpUrl`). Сейчас `rsvpUrl` — заглушка (`#`); замените на реальную форму или чат, когда она будет готова.
- `src/components/sections/` — семь экранов приглашения (обложка, место, программа, трансфер, дресс-код, пожелания, подтверждение).
- `src/components/decor/` — SVG-декор: ботанические ветки, винтажная рамка, арка.
- `src/components/layout/` — обёртки страницы и секций (в том числе плавное появление секций при скролле).
